Resume Analyzer Pro (RAP)
Overview

Resume Analyzer Pro (RAP) is an AI powered web application designed to automate the resume screening process. The system analyzes resumes,
extracts important information, compares resumes with job descriptions, and provides intelligent insights such as ATS scores, compatibility ratings, and skill gap analysis.
The project helps recruiters, placement cells, and job seekers make faster and more data driven decisions during the hiring process.
